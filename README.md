**How to get AuthorizationID**
1. Open Chrome/Edge
2. Open Discord App
3. Open Chrome Dev Tools
4. Go to "Network" Tab
5. Click on any Discord server
6. On your left you should see 1000 files loading, click on the one "messages?limit=50"
7. Now to your right click on Headers (Should be opened by default)
8. Scroll down and copy the value at "authorization:"
